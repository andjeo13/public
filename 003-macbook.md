# ADR-003: Выбор MacBook

- **Статус:** принято
- **Дата:** 2026-04-05

## Контекст

Текущий ноутбук — MacBook Pro 13" (2020, MacBookPro16,2):
- Intel Core i5 Quad-Core, 2 GHz
- 16 ГБ RAM
- 1 ТБ SSD (Apple SSD AP1024N)
- Intel Iris Plus Graphics
- macOS 26.2 (Tahoe)

### Требования

- Экран 11–14"
- Процессор Apple Silicon (M-серия)
- Минимальный нагрев
- Большая автономность
- HDMI не нужен

### Сценарии использования

**Работа:** почта, файлы, презентации, AI-инструменты (Claude, ChatGPT, Copilot)
**Дома:** работа с AI, пет-проекты (код, Docker), домашняя автоматизация (WB, MQTT, Home Assistant), просмотр видео

**Почему не Snapdragon X / Intel:**
- Docker на ARM Windows менее стабилен, чем на macOS
- Эмуляция x86 не идеальна
- Все имеют вентилятор, батарея хуже MacBook Air
- Переход с macOS на Windows — потеря экосистемы и настроек

## Решение

**MacBook Air 13" M5, 24 ГБ / 1 ТБ (~$1,499 / ~175k ₽)**

### Обоснование

1. **Безвентиляторный** — нулевой шум, минимальный нагрев (ключевое требование)
2. **18 ч батареи** — лучший показатель среди безвентиляторных ноутбуков
3. **1.23 кг** — самый лёгкий вариант среди Apple
4. **Wi-Fi 7 + Bluetooth 6** — новее чем у Pro 14" M5
5. **Бесшовный переход** с текущего MacBook (macOS, данные, Homebrew, настройки)
6. **Docker** работает нативно на ARM, весь стек (WB, MQTT, Python, HA) совместим
7. **24 ГБ RAM** — на текущем MBP 16 ГБ уже уходит в swap (Claude Code + Docker + браузер), 24 ГБ даёт запас на весь срок жизни
8. **1 ТБ SSD** — аналог текущего объёма
9. **Цена** — на 35k ₽ дешевле Pro 14" M5 с аналогичной RAM

### Компромиссы

- Экран 60 Hz / 500 нит (vs 120 Hz / 1600 нит у Pro) — приемлемо для рабочих задач
- Тротлинг при длительной нагрузке (нет вентилятора) — для пет-проектов не критично
- 2 порта USB-C (vs 3 у Pro) — достаточно с хабом
- Нет HDMI/SD — не нужен
